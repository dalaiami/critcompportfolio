SOUVEILLANCE TOOL: ON SET

Description

This project was made with a nosy mind. I wanted to know where filming was happening. Apparently, all around.

Design Process

I started with finding an interesting API, and then moved on to designing around that.

Reflection

I had issues going into the API itself from the sketch, but I was able to set up a sketch that shows where filming is taking place. The locations are randomized from 2020-2021.

Credits

This project was created using NYC Open Data on filming permits.
